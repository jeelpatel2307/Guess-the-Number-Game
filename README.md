# Guess the Number Game

This is a simple Java program for playing the Guess the Number game. In this game, the computer selects a random number between 1 and 100, and the player needs to guess that number.

## How to Play

1. Clone this repository or download the source code.
2. Compile the Java source file (`GuessTheNumber.java`) using a Java compiler.
3. Run the compiled Java program (`GuessTheNumber.class`).
4. Follow the instructions to guess the number.

## Gameplay

- The game will prompt you to enter your guess.
- If your guess is higher than the secret number, the game will tell you to try a lower number.
- If your guess is lower than the secret number, the game will tell you to try a higher number.
- Keep guessing until you guess the correct number.

## Features

- Random number generation ensures each game is unique.
- Keeps track of the number of attempts taken to guess the correct number.

Have fun playing the game!
